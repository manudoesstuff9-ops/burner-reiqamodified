──────────────────────

  • KADIS → training_data/kadis/

  • COCO unlabeled2017 → training_data/coco_unlabeled/
  • COCO test2015 → training_data/coco_test/

──────────────────────────────
Windows PowerShell:

Linux/macOS:

Step 4: Watch training happen
──────────────────────────────
The script will automatically:
  ✓ Create/activate virtual environment

  ✓ Create necessary directories
  ✓ Start training for 200 epochs
  ✓ Save checkpoints every epoch

Estimated training time: 1-2 hours on modern GPU (24-48 hours on CPU)

10. TROUBLESHOOTING QUICK REFERENCE
================================================================================

Problem: "No images found" error
Solution: Check that files are extracted directly in the folder
  cd training_data/kadis
  dir (or ls on Mac/Linux)  ← Should show image files directly

Problem: "ModuleNotFoundError: No module named 'torch'"
Solution: Let the script install it (run MASTER_SETUP_AND_RUN.ps1 again)
          Or manually: pip install torch torchvision

Problem: "Permission denied" on .sh script
Solution: chmod +x MASTER_SETUP_AND_RUN.sh

Problem: "CSV file not found" error
Solution: Make sure you're in the correct directory
  cd c:\Users\mohit\Desktop\reiqamodified\burner-reiqamodified

Problem: Training is very slow
Solution: This is normal! First epoch takes time
          Monitor: GPU usage, CPU usage, memory usage
          Should stabilize after batch 2-3

Problem: Out of memory (OOM) error
Solution: Reduce batch size in the script
  Change: --batch_size 128 → --batch_size 64
