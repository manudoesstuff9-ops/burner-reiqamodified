❓ WHAT IF...
================================================================================

Q: The script says "No images found in training_data"
A: You haven't downloaded/copied the datasets yet!
   → Download them and extract to the correct training_data subfolder
   → See DATASET_ORGANIZATION_GUIDE.txt for detailed instructions

Q: Error about GPU/CUDA not found
A: This is OK! The script will automatically use CPU instead
   → Training will be slower but still work
   → Install PyTorch with GPU support if you have NVIDIA GPU

Q: "ModuleNotFoundError" for torch or other packages
A: The script will install requirements automatically
   → Make sure you have internet connection
   → Check requirements.txt has all packages listed

Q: Training seems stuck/very slow
A: This is normal for first epoch with large datasets
   → Dataset loading and image processing take time
   → GPU usage should ramp up after first 2-3 batches
   → Monitor GPU usage: nvidia-smi (if you have NVIDIA GPU)

Q: I only have KADIS-700K dataset
A: That's fine! Just download that one dataset
   → Extract to training_data/kadis/
   → CSV already has KADIS entries (~200 images)
   → Run the script - it will train only on KADIS
